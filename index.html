Credits & acknowledgements

Gearbox Solver is a single-file web app. It stands on the shoulders of these open-source libraries and references.

Libraries
  • • Three.js — 3-D rendering (r160, MIT)
  • • Matter.js — 2-D rigid-body collision physics (MIT)
  • • poly-decomp.js — concave polygon decomposition for Matter.js (MIT)
  • • Delaunator — fast Delaunay triangulation used for the FEA mesh (ISC)
  • • Tailwind CSS — UI styling (MIT)
Geometry & engineering references
  • • Involute tooth profile from the gears.scad OpenSCAD library (port of the classical ev(r,ρ) polar involute).
  • • DIN 867 / DIN 58400 gear proportions and root-clearance conventions.
  • • Lewis bending-stress form factor (AGMA 20° full-depth approximation).
  • • Hertzian contact-stress closed-form solution for gear-pair line contact.
FEA implementation

Hand-rolled 2-D plane-stress constant-strain-triangle (CST) solver with sparse assembly, Jacobi-preconditioned conjugate-gradient, and rotational-symmetry tensor superposition. Not derived from any library — written from standard textbook formulations.

Colormaps

Thermal / viridis / inferno presets inspired by the matplotlib perceptually-uniform colormap family.
